# Spring Cloud

### Cloud Stream

Компонент для для обеспечения связи с брокерами сообщений. Легко интегрируется с RabbitMQ, Kafka, Amazon Kinesis,
PubSub, RocketMQ и т.д.

---

### Feign

Простой и гибкий http-клиент. Использует интерфейсы аннотированные @FeignClient чтобы генерировать API запросы и мапить
ответ на Java классы. Он шлет http запросы другим сервисам.

---

### Eureka Server

Это Discovery Server (обнаружение сервисов) для ваших микросервисов. Клиентские приложения могут самостоятельно
регистрироваться в нем, а другие микросервисы могут обращаться к Eureka Server для поиска необходимых им микросервисов.
Для этого сервис должен быть помечен как @EnableEurekaClient, а сервер @EnableEurekaServer. Содержит такую информацию
как IP-адрес и порт микросервиса.

---

### Gateway
Сервис для маршрутизации входящих запросов между микросервисами и балансировки между экземплярами
микросервисов. Заменяет устаревший Zuul и поддерживает реактивность и WebSockets.

---

### Config Server|Client
Компонент для хранения и обслуживания распределенных конфигураций. Идеально версионировано под
управлением версией Git и может быть изменено во время выполнения приложения. Мы должны использовать аннотации
@EnableConfigServer и @EnableConfigClient.

---

### Ribbon
Это балансировщик нагрузки. Из коробки он интегрирован с механизмом Service Discovery.
Предоставляет: отказоустойчивостьб, Load balancing, поддержка нескольких протоколов (HTTP, TCP, UDP) в асинхронной и
реактивной модели, кеширование и т.д.
Eсть 2 вида LoadBalancer: Server side и Client-Side.

---

### Bus
Компонент доставки и обновления конфигураций через брокеры Kafka или AMQP.

---

### Cloud Circuit Breaker
Компонент для повышения отказоустойчивости сервисов, путем временного прерывания отправки
запросов

---

### Sleuth
Реализует трассировку и логирование запросов для анализа проблем в распределенной системе.

---

### Hystrix
Это библиотека задержек и отказоустойчивости, это имплементация паттерна Circuit Breaker.
